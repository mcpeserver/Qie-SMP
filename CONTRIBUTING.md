# Panduan Kontribusi

Terima kasih banyak atas ketertarikan Anda untuk berkontribusi pada pengembangan website **QIE SMP**! Kontribusi dari komunitas sangat membantu kami untuk terus menyempurnakan kualitas website ini.

---

## 🛠️ Bagaimana Anda Bisa Membantu?

Anda dapat berkontribusi dalam beberapa bentuk berikut:
1. **Melaporkan Masalah (Bug Reporting)**: Menemukan link yang rusak, teks bertabrakan, atau fungsi salin IP yang bermasalah? Laporkan melalui tab *Issues* di GitHub.
2. **Saran Fitur**: Ada usulan desain atau tata letak kartu yang lebih menarik? Silakan sampaikan ide Anda.
3. **Mengirimkan Perbaikan (Pull Request)**: Anda dapat memperbaiki kode secara langsung dan mengirimkan pengajuan penggabungan kode.

---

## 🚀 Alur Mengirimkan Kontribusi (Pull Request)

Silakan ikuti langkah-langkah di bawah ini untuk mengirimkan perubahan kode Anda:

1. **Fork Repositori**: Buat salinan repositori ini ke akun GitHub Anda pribadi.
2. **Buat Cabang Baru (Branch)**:
   ```bash
   git checkout -b fitur/fitur-keren-anda
   ```
3. **Lakukan Perubahan Kode**: Pastikan kode tetap rapi, menggunakan TypeScript dengan benar, dan mengikuti pola warna Minecraft Nature yang telah ditetapkan.
4. **Lakukan Kompilasi & Pengujian**:
   Jalankan perintah pengujian untuk memastikan tidak ada syntax error:
   ```bash
   npm run build
   ```
5. **Simpan Perubahan (Commit)**:
   ```bash
   git commit -m "Tambah: Detail animasi baru pada kartu peraturan"
   ```
6. **Kirim Perubahan (Push)**:
   ```bash
   git push origin fitur/fitur-keren-anda
   ```
7. **Buat Pull Request**: Buka repositori asli dan ajukan *Pull Request* baru dari cabang Anda. Kami akan segera meninjau pekerjaan luar biasa Anda!
